# 🏎️ F1 Race Predictor — Multi-Model ML System

Система прогнозирования результатов гонок Формулы-1 с гибридным ансамблем ML-моделей, предсказанием сходов (DNF) и **уникальным NLP-анализом новостей** через FinBERT.

---

## ✨ Почему это интересно

Большинство моделей предсказывают F1, опираясь только на цифры.
Здесь добавляется **контекст из новостей**, который реально влияет на результат:

* 📰 Анализируются свежие статьи (Motorsport, The Race)
* 🤖 FinBERT определяет тональность (позитив / негатив)
* 🧠 Модель корректирует прогноз с учетом информационного фона

👉 В итоге модель понимает не только *что было*, но и *что происходит сейчас*.

---

## 🧠 Архитектура модели

### 1. Основная модель (позиции)

Ансамбль двух моделей:

* **RandomForestRegressor** (основа стабильности)
* **XGBoost** (адаптивность)

| Модель       | Вес    |
| ------------ | ------ |
| RandomForest |  65% |
| XGBoost      | 35% |

📌 Результат: предсказанные значения сортируются → позиции 1–22

---

### 2. Модель сходов (DNF)

* Logistic Regression
* Балансировка классов
* Временное взвешивание данных



---

### 3. NLP-модуль

Pipeline:

1. Парсинг новостей
2. Суммаризация
3. Контекстное расширение
4. Анализ через FinBERT
5. Усреднение sentiment

📈 Влияние на прогноз:

```
final_score = base_score × (1 + sentiment × 0.15)
```

---

### 4. Feature Engineering

Создается **35+ признаков**:

* История выступлений
* Форма пилота
* Взаимодействие с командой
* Погодные условия
* Особенности трассы
* Кросс-фичи (температура × стратегия и т.д.)

---

## 🧠 Сравнение с экспертами

Важно учитывать контекст:

* Большинство экспертов и аналитиков:

  * дают прогноз только **на топ-3**
  * не формируют полный порядок финиша
* Модель в большинстве случаев лучше предсказывает подиум - https://www.youtube.com/watch?v=2eEwtej6Vlc&t=1081s

👉 В этом смысле модель:

* **сильнее типичных экспертных прогнозов** (за счёт полного ранжирования и системности)
* но пока **уступает продвинутым AI-решениям**, таким как суперкомпьютер GiveMeSport

---

## ⚠️ Ограничения сравнения

* Букмекеры предсказывают **вероятности**, а не позиции
* Суперкомпьютеры используют:

  * проприетарные данные
  * продвинутые симуляции
  * большие вычислительные ресурсы

👉 Моя модель использует:

* только **открытые данные**
* **NLP-анализ новостей**
